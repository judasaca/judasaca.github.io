सिफारिशें:

  • HTML 5: एक शब्दावली और HTML और XHTML के लिए एसोसिएटेड एपीआई
  • एक्सएचटीएमएल ™ 1.0: एक्स्टेंसिबल हाइपरटेक्स्ट मार्कअप लैंग्वेज
  • एक्सएचटीएमएल मॉड्यूलर
  • एक्सएचटीएमएल 1.1
  • एक्सएचटीएमएल बुनियादी
  • एक्सएचटीएमएल छाप

  • HTML 5.2
